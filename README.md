# 微分ツール

このリポジトリは、対数微分法や普通の微分をブラウザ上で計算・表示するシンプルなWebツールです。

## 含まれるファイル

- `index.html` - GitHub Pages からアクセスするためのエントリポイント（`differential.html` へリダイレクトします）
- `differential.html` - 実際のツール画面
- `differential.css` - スタイル定義
- `differential.js` - 計算ロジックと表示制御

## ローカルで見る

1. `index.html` をブラウザで開く
2. または `differential.html` を直接開く

## GitHub Pages で公開する場合

1. GitHub で新しいリポジトリを作成
2. このフォルダー内のファイルをリポジトリに追加してコミット
3. GitHub Pages を有効化し、公開元を `main`（もしくは `master`）ブランチのルートに設定
4. `https://<ユーザー名>.github.io/<リポジトリ名>/` にアクセス

## 例

`differential.html` にアクセスするとツールを使えます。

## 注意

- GitHub のリポジトリ内では、HTML ファイルはそのまま表示されません。GitHub Pages を使うとブラウザ上で動作するページとして閲覧できます。
